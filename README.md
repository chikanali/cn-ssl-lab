# cn-ssl-lab

本仓库用于归档和发布多个独立 HTML 页面，内容涉及 SSL/TLS 相关的学习、测试与参考场景。

## 项目简介

这是一个以 HTML 页面为主要内容的归档仓库，主要面向 SSL/TLS 领域的技术资料、实验页面与辅助工具。所有页面均为独立文件，不依赖特定域名或网站，适合用于本地查看、离线存档或简单部署。

## 目录结构

- `/` — 根目录：存放各个独立 HTML 页面（如 index.html、test-page.html 等）
- `/assets/` — 样式、脚本、图片等公共资源（如有）
- `/docs/` — 相关文档或说明（可选）

随着仓库更新，目录结构可能会调整，请以实际文件为准。

## 页面归档说明

- 每个 HTML 页面均为独立的完整文件，可直接在浏览器中打开使用。
- 页面内容主要围绕 SSL/TLS 协议、证书、加密算法等基础知识或测试场景。
- 部分页面可能包含交互式控件（如输入框、按钮），用于验证或演示特定概念。
- 所有页面均不包含任何具体域名、网址或商业推广信息。

## 维护说明

- 本仓库会不定期新增或更新 HTML 页面。
- 更新时可能涉及文件结构调整或旧页面移除，建议定期关注仓库动态。
- 欢迎提交 Issue 或 Pull Request 进行内容补充或问题反馈。
- 本仓库仅作归档与展示用途，不提供任何形式的服务或承诺。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开任意 HTML 文件即可查看。
3. 如需部署到服务器，请确保服务器支持静态文件托管。
